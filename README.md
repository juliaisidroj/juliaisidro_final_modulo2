## EJERCICIO EVALUACION MODULO 2

En este ejercicio vamos a crear bases de datos, SAKILA y PELICULAS, y vamos a realizar distintas consultas en ellas

Nos encontramos con dos ejercicios:
1. El primero de ellos, trata de hacer consultas sobre la base de datos de Sakila.
   Los datos de dicha base de datos nos los han proporcionado (los podemos ver en "sakila_bd" y "sakila_data"), por lo tanto, simplemente hemos copiado y ejecutado su contenido en MYSQL Workbench, y a partir de ahí, hemos hecho distintas consultas, las cuales las podemos encontrar en el documento "MYSQL_SAKILA"

2. El segundo de ellos trata de crear una base de datos, obteniendo dicha informacion del siguiente API "https://beta.adalab.es/resources/apis/pelis/pelis.json", para, posteriormente, hacer consultas sobre ella.
   En este caso, dado que podiamos utilizar Phyton y MYSQL Workbench, hemos decidido que, en primer lugar, vamos a obtener los datos del API y crear la base de datos desde Python, para así poder explorar la conexion entre Phyton y MYSQL Workbench. Posteriormente, haremos las consultas en MYSQL Workbench, dado que, a nivel personal, lo considero una practica mas sencilla.

   La creacion de la base de datos a partir de un API la podemos encontrar en "Creacion_BBDD_Phyton".
   Las consultas realizadas las podemos encontrar en "MYSQL_PELICULAS"
