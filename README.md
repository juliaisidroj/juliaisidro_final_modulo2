## EJERCICIO EVALUACION MODULO 2

En este ejercicio vamos a crear bases de datos, SAKILA y PELICULAS, y vamos a realizar distintas consultas en ellas

Nos encontramos con dos ejercicios diferentes:
1. El primero de ellos trata de hacer consultas sobre la base de datos de Sakila.
   Los datos de dicha base de datos no lo han proporcionado (los podemos ver en "sakila_bd" y "sakila_data"), por lo tanto, siemplemente hemos copiado y ejecutado su contenido en MYSQL Workbench, y a partir de ahí hemos hecho distintas consultas, las cuales aparecen en el documento "MYSQL_SAKILA"

2. El segundo de ellos trata de crear una base de datos, obteniendo dichos datos del siguiente API "https://beta.adalab.es/resources/apis/pelis/pelis.json", y nuevamente, hacer consultas sobre ello.
   En este caso, dado que podiamos utilizar Phyton y MYSQL workbench, hemos decidido que en primer lugar, vamos a obtener los datos de API y crear la base de datos desde Python, para así poder explorar la conexion entre Phyton y MYSQL Workbench, para posteriormente, hacer las consultas en MYSQL Workbench, dado que a nivel personal, me parece mucho mas sencillo la realizacion.

   La creacion de la base de datos a partir de un API lo podemos encontrar en "Creacion_BBDD_Phyton". LAs consultas realizadas las podemos encontrar en "MYSQL_PELICULAS"
